Сума елементів масиву
Дано масив чисел. Використовуйте reduce, щоб знайти суму всіх елементів масиву.
input: [1, 2, 3, 4] / output: 10; 

console.log([1, 2, 3, 4].reduce((accFin, currValue) => {
  return accFin + currValue;
}, 0));
console.log([1, 2, 3].reduce((acc, val) => acc + val, 0));
console.log([3, 2, 3].reduce((a, b) => a + b));
