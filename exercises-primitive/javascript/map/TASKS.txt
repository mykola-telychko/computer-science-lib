Квадрати чисел. Створіть масив чисел.
Використайте map, щоб повернути новий масив, де кожен елемент 
є квадратом відповідного числа з оригінального масиву.
console.log( [12, 4, 56, 6, 3, 9, 23, 6, 87].map(i => { return i * i; }) );


Рядки в верхній регістр. Створіть масив рядків (слів).
Використайте map, щоб перетворити кожен рядок у новий масив, 
де всі літери будуть у верхньому регістрі.
console.log(['ssdf', 'dfgd', 'efbefb', 'qscqsc', 'ppp'].map(i => {return i.toUpperCase()}));


Перетворення об'єктів
Маєте масив об'єктів, де кожен об'єкт містить інформацію про користувача ({ name: '...', age: ... }).
Використайте map, щоб створити новий масив, який містить лише імена користувачів (масив рядків).
console.log([{ name: 'Mark', age: 34 },
             { name: 'Will', age: 45 }, 
             { name: 'Robin', age: 33 }, 
             { name: 'Saba', age: 32 }].map(i => {return i.name} ));

Додавання поля до об'єкта
Маєте масив об'єктів товарів, де кожен об'єкт має поля name та price.
Використайте map, щоб створити новий масив об'єктів, де до кожного об'єкта додається 
нове поле isAvailable зі значенням true.
console.log([{ name: 'Bread', price: 34 },
             { name: 'Milk', price: 45 }, 
             { name: 'Meat', price: 33 }, 
             { name: 'Salt', price: 32 }]
.map(i => {return {name: i.name, price: i.price, isAvailable: true}} ));

----------------2--------------
const products = [
    { name: 'Bread', price: 34 },
    { name: 'Milk', price: 45 },
    { name: 'Meat', price: 33 },
    { name: 'Salt', price: 32 }
];
const updatedProducts = products.map(item => ({
    ...item,
    isAvailable: true
}));
console.log(updatedProducts);
----------------2--------------
----------------3--------------
const addField = (array, key, value) => array.map(item => ({
    ...item,
    [key]: value
}));
const updatedProducts = addField(products, 'isAvailable', true);
console.log(updatedProducts);

Обчислення загальної вартості кошика.
Маєте масив об'єктів, що представляють товари в кошику. Кожен об'єкт містить price і quantity.
Використайте map, щоб створити новий масив, де кожен елемент — це загальна вартість одного товару 
(price * quantity). Потім використайте reduce для обчислення загальної суми.
# math 
console.log([{ name: 'Bread', price: 34, quantity: 3},
             { name: 'Milk', price: 45, quantity: 5}, 
             { name: 'Meat', price: 33, quantity: 9}, 
             { name: 'Salt', price: 32, quantity: 2}]
.map(i => {return i.price * i.quantity} ));
